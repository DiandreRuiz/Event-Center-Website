This project is meant to showcase mastery of fundamental HTML & CSS concepts.

No Javascript was used in the creation of the website, only vanilla HTML & CSS.

The theme of the website is an event center based in New York City that hosts upscale business gatherings, weddings, receptions etc.

You will notice small changes to the mockups located in the mockups/ directory, these were style changes or last minute alterations meant to enhance
the user experience of the website.
